## 6. Boundary check

### Fidelity
- Does it still answer the same question?
- Are the key logic relations preserved?
- Are scope and uncertainty preserved?

### Clarity
- Can a first-time reader tell what reality this is about?
- Are abstract nouns reduced where possible?
- Are actions, relations, and situations visible?

### Evidence visibility
- Are important factual claims traceable?
- Did explanation get mixed into rewrite?
- Were unsupported claims added?

### Language discipline
- Did the rewrite become too soft?
- Did it become too chatty?
- Did it replace one jargon with another?
