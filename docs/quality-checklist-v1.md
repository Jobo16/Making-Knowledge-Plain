# 质量校验表 v1

每完成一次复述，都应该用这份表检查质量。

## 一、核心问题是否没变

- 原文在回答什么问题？
- 我的复述还在回答同一个问题吗？
- 我有没有把问题偷偷改成更容易说的版本？

## 二、逻辑关系是否没变

检查以下内容有没有被改坏：

- 因果
- 条件
- 转折
- 对比
- 递进
- 范围
- 程度
- 不确定性

## 三、信息是否凭空增加

- 我有没有把自己的理解当成原文意思塞进去？
- 有没有哪句话无法回指到原文依据？
- 哪些内容其实属于“解释”或“判断”，却被我混进了“复述”？

## 四、语言有没有变空

- 我是不是为了好懂，把原文真正锋利的东西磨平了？
- 我是不是把结构性的判断翻成了情绪化安慰？
- 我是不是只留下常识，没有留下推进？

## 五、语言有没有变胖

- 我是不是去掉了术语，却加了大量口水话？
- 我是不是把一句清楚的话拖成了一段散话？
- 我是不是增加了不必要的修辞？

## 六、读者是否能直接进入

- 读者第一次读，能否知道这是在说什么现实？
- 读者能否知道它在解释什么问题？
- 读者是否需要先理解一套圈内语言才能进入？

## 七、是否保留了原文力度

- 原文中的张力、冲突、批判、诊断是否还在？
- 我有没有把它写得更温和、更空、更安全？
- 我有没有为了顺口，牺牲原文的判断强度？

## 八、三层内容是否分开

以下三层是否明确区分：

- 忠实复述
- 辅助解释
- 个人判断

如果没有分开，读者很容易误以为你的解释就是原文意思。

## 九、最终判断

一次合格的复述，应同时满足：

- 保真度高
- 理解阻力低
- 无擅自发挥
- 无废话增殖
- 可回指原文

## 十、一句话标准

不是更简单，而是更透明；不是更松散，而是更可进入。
